# メール作成の使用例

## テンプレート

```
Opus 4.8 Fable-like Modeでメール文を作成してください。

目的：
{メールの目的}

相手：
{相手の立場や関係}

伝えたい内容：
{要点}

条件：
- 丁寧なビジネス文
- 失礼がない
- 回りくどすぎない
- 必要な依頼事項を明確にする
- そのまま送れる形にする
```

## 記入例：納期遅延の連絡

```
Opus 4.8 Fable-like Modeでメール文を作成してください。

目的：
部品の入荷遅れにより、装置の納品が1週間遅れることを連絡し、了承を得る

相手：
取引先の購買担当者（付き合いは3年、関係は良好）

伝えたい内容：
- 制御基板の入荷が遅れ、納品予定が7/15から7/22になる
- 遅延の原因はこちらの発注ミスではなく部品メーカーの供給遅れ
- 代替案として、先に本体のみ納品し基板は後日交換対応も可能
- どちらがよいか返答がほしい

条件：
- 丁寧なビジネス文
- 失礼がない
- 回りくどすぎない
- 必要な依頼事項を明確にする
- そのまま送れる形にする
```

## 期待される動き

1. 件名案を含め、そのまま送れる全文で出力される
2. 謝罪・事実・代替案・依頼（返答期限）が明確に分かれた構成になる
3. 責任の所在について断定しすぎる表現（相手に失礼な言い訳と取られる表現）は避けられる
4. 英語メールが必要な場合は、条件に「英語版も併記」と書けば日英併記になる
